# Automação Linux - Notebook pessoal

Criei este repositório para guardar os scripts que uso no dia a dia no meu notebook, automatizei tarefas repetitivas para ser mais objetivo e gastar menos tempo.

O que os scripts fazem?

- - -"clean.sh"- - -
Ao invés de digitar comando por comando, esse script faz a limpeza por mim.
Ele atualiza o sistema, remove arquivos temporários e garante que meus Flatpaks estejam atualizados, é um comando "faxina" antes de começar ou finalizar o dia.

- - -"study.sh"- - -
Esse script prepara meu ambiente de estudo. Ele checa se o VS Code já está rodando,para não abrir janelas duplicadas, abre o firefox com a aba do GitHub e do Gemini e por fim abre o gerenciador de arquivos direto na minha pasta de projetos. Perfeito para não gastar tempo na faculdade ou estudando em casa.

---"buds.sh"---
Economizo meu tempo ao tentar conectar meu fone bluetooth no notebook, apenas ao digitar "buds" no terminal, e ele faz a conexão automaticamente para mim.

Para facilitar, criei aliases no meu Ubuntu. Assim apenas ao digitar o nome dos arquivos no terminal, eles já são executados.
